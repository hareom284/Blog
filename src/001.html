Laravel ကို သုံး ပြီး web application တည် ဆောက် ရ တွင် အောက် ပါ အ ကျိုး ကျေးဇူး ရ ရှိ နိုင် ပါ သည်။

  • Application ကို design အ တွက် စဉ်း စာ စ ရာ မ လို ပဲ laravel မှာ ပါ ‌သော build in components ယူ သုံး လို့ ရ ပါ တယ်။
  • Laravel သည် namespace and interface သုံး သော ‌ကြောင့် data တွေ့ manage လုပ် ရ တာ အ ရမ်း လွယ် ကူး သွား ပါ တယ်
  • Laravel ကို သုံး ပြီး ရေး ချင်း က files structure က ရှုပ် မ နေ ပဲ application ကို တစ် ချိန် maintain ပြန် လုပ် ရင် အ ရမ်း လွယ် ကူး သွား ပါ တယ်။

Before installation Requirement

  • PHP version က 7.2.5 or ထက် ကျော် ဖို့ လို တယ်

Laravel လေ့လာမယ်ဆို လုပ်အပ်မယ်အချက်ကတော့

HTML နဲ့ familiar ဖြစ် ထား ဖို့ လို ပါ ၊ Little bit Ajax (but optional)၊PHP pure ကို နာ လည်း ထား ခြင်း ၊ MVC က ဘယ် လို အ လုပ် လုပ် လဲ ဆို တာ သိ ထား ခြင်း အား ဖြင်း laravel လေ့ လာ တယ် အ ခါ ပို ပြီး လွယ် ကူ လျင် မြန် လာ ပါ မယ်။

Laravel basic တွေ့ ကိုကျွန် တော် cover လုပ် သွား မယ် outline တွေ့ က တော့ အောက် မှာ ဖော် ပြ ထား ပါ တယ်
